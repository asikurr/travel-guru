# Travel Guru 

### Details

A Complete React Js Website for a travel booking company, where user can book their destination ticket based on favorable date.

### Languages and Tools :

- React JS
- Google Map
- React-Bootstrap
- Firebase Auth
- Firebase Hosting
- Firebase Tool
- VS Code
- Git
- Github
